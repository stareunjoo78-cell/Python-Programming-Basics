# Python-Programming-Basics
Fundamentals of Python programming for Data Science and Machine Learning applications.

# [Course] Python Basic: 실무 프로그래밍 15주 완성

> 본 과정은 파이썬 기초 문법부터 데이터 분석 실무까지 단계별로 학습하여 현업에 즉시 적용 가능한 역량을 배양하는 것을 목표로 합니다.

## 주차별 상세 커리큘럼

### [1단계] 파이썬 기초 및 환경 구축
* **1주차: 개발 환경 설정 및 프로그래밍 입문**
    * Python 설치, Jupyter Notebook 환경 설정
    * Hello World 출력 및 파이썬의 주요 특징 이해
* **2주차: 자료형(Data Types)과 변수**
    * 숫자형, 문자열(Indexing/Slicing), 불(Bool) 자료형 학습
    * 변수 선언 및 데이터 타입 변환(Casting) 실습
* **3주차: 제어문 1 (조건문)**
    * `if`, `elif`, `else`를 활용한 논리 구조 설계
    * 비교 연산자와 논리 연산자 실무 적용
* **4주차: 제어문 2 (반복문)**
    * `for`, `while` 루프를 이용한 반복 작업 자동화
    * `range()` 함수와 중첩 루프(Nested Loop) 활용 실습

### [2단계] 데이터 구조 및 함수 응용
* **5주차: 자료구조 심화 (컨테이너 자료형)**
    * List, Tuple, Dictionary, Set의 특징과 주요 메서드 활용
    * 데이터 수집 및 관리를 위한 최적의 자료구조 선택 전략
* **6주차: 함수(Function)와 모듈**
    * 매개변수, 반환값, 가변 인자(`*args`, `**kwargs`) 이해
    * `lambda` 표현식 및 내장 함수(`map`, `filter`) 활용
* **7주차: 클래스(Class)와 객체지향 프로그래밍**
    * 객체의 개념, 생성자(`__init__`), 메서드 구현
    * 상속(Inheritance)과 캡슐화 기초 개념 이해
* **8주차: 중간고사**
    * 파이썬 기초 문법 및 알고리즘 구현 테스트 (실기/필기)

### [3단계] 데이터 전처리 및 분석 기초
* **9주차: 예외 처리 및 파일 입출력(I/O)**
    * `try-except`를 이용한 런타임 오류 제어
    * 텍스트 파일(.txt) 및 CSV 파일 읽고 쓰기 실습
* **10주차: 데이터 분석 필수 라이브러리 1 (NumPy)**
    * 넘파이 배열(Array) 생성 및 다차원 연산 기초
    * 통계 함수 활용 및 불리언 인덱싱을 통한 데이터 필터링
* **11주차: 데이터 분석 필수 라이브러리 2 (Pandas)**
    * Series와 DataFrame 구조 이해
    * 데이터 로드, 인덱싱, 정렬 및 기초 기술통계 산출
* **12주차: 데이터 전처리 실무 (Pandas 응용)**
    * 결측치(Missing Value) 처리 및 데이터 병합(`merge`, `concat`)
    * 조건에 따른 데이터 그룹화(`groupby`) 및 집계 실무 연습

### [4단계] 데이터 시각화 및 최종 프로젝트
* **13주차: 데이터 시각화 기초 (Matplotlib/Seaborn)**
    * 주요 그래프(Line, Bar, Scatter, Histogram) 구현
    * 한글 폰트 설정 및 시각화 결과 리포트 작성법
* **14주차: 종합 실무 실습 (Mini Project)**
    * 공공데이터(CSV)를 활용한 데이터 분석 프로세스 총정리
    * 데이터 정제부터 인사이트 도출, 시각화까지 전 과정 구현
* **15주차: 기말고사**
    * 최종 프로젝트 결과물 발표 및 개별 평가
